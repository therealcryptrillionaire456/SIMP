## 🚀 How to Use

### Quick Start:
```python
from integration.brp_enhanced_framework import BRPEnhancedFramework

# Start in defensive mode
framework = BRPEnhancedFramework(mode='defensive')

# Submit security events
framework.submit_event({
    'source': 'attacker.com',
    'event_type': 'threat',
    'content': 'Attack detected'
})

# Run defensive scan
framework.run_defensive_scan()

# Test offensive capabilities (authorized only)
framework.test_offensive_capability("pentagi_scan", "target.test")
```

### Operation Modes:
1. **Defensive** (Default): Monitor and protect
2. **Offensive**: Execute authorized offensive operations
3. **Hybrid**: Combined defense with opportunistic offense
4. **Intelligence**: Analysis, planning, knowledge management
